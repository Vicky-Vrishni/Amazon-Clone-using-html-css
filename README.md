# Amazon Clone

This is a simple Amazon clone made using HTML and CSS. I created this project to practice frontend development and understand how real websites are structured.

## Features
- Responsive layout
- Navbar with search bar
- Hero banner section
- Product grid layout
- Footer section

## Tech Stack
- HTML
- CSS (Flexbox & Grid)

## Project Structure
amazon-clone/
- index.html
- style.css

## Note
This is just a frontend project for learning purposes. It does not have any backend functionality.
